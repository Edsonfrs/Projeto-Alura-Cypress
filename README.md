# Cypress – Automatizando Testes E2E

## 📌 Sobre o projeto

Este repositório contém um projeto de **automação de testes end-to-end (E2E)** utilizando **Cypress**, desenvolvido como parte de um **treinamento da Alura**, integrante da **Carreira de QA**.

O objetivo principal é aplicar, na prática, conceitos fundamentais de automação de testes, organização de cenários, geração de relatórios e boas práticas no uso do Cypress em projetos reais.

## 📚 Conteúdos abordados no treinamento

Durante o desenvolvimento deste projeto, foram trabalhados os seguintes tópicos:

- Configuração de um projeto de testes Cypress via interface gráfica
- Utilização dos principais métodos e funções do Cypress
- Captura e utilização de seletores de elementos para automação de testes
- Geração de documentação e relatórios de testes com o **Mochawesome**
- Uso do **Cypress Cloud** para compartilhamento de resultados em nuvem
- Criação de dublês de teste (mocks e stubs)
- Utilização de massa de dados na automação de testes
- Criação de comandos personalizados para melhor organização e reutilização de código
